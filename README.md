# data-portal
React/Redux data-portal skeleton

Start in Debug Mode:
```
npm install
npm start
```

Start with mock server:
```
npm run mock
```

Start in Production Mode:
```
npm run build
npm install -g serve
serve -s build
```

# DAF Open Map
React Open Map Component with OpenStreetMap and react-leaf.

How to start:
```
cd react-open-map
npm install
yarn storybook
```
